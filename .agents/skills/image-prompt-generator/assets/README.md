# Image Prompt Generator Assets

Drop high-quality reference images here for the `image-prompt-generator` skill.

Recommended structure:

- `hero-warehouse-good.png`
- `section-logistics-illustration-good.png`
- `og-card-minimal.png`

Use this folder to store visual examples of the kinds of composition, palette, lighting, and style the skill should keep in mind.

If your agent tooling can access workspace files, the skill can refer to these filenames when describing or comparing reference images.
